Datasets in R

So far, we have considered two ways of importing datasets into R. In Lab 2, we discussed creating dataframes in R by using the function data.frame() function and by typing the data values in a vector. In Lab 3, we discussed how to import data from a CSV file and create a dataframe in R. In this lab, we discuss datasets available in the.

R comes with several datasets built into its base installation, which are available for users to explore and use for various data analysis and statistical tasks. These datasets are typically used for practice, examples, and teaching purposes. Here are some of the commonly used datasets that come with R by default:

    1. Iris Dataset: Perhaps one of the most famous datasets, it contains measurements of iris flowers from three different species: setosa, versicolor, and virginica. It’s often used for classification tasks.
    1. mtcars Dataset: This dataset provides information about various car models, including attributes such as miles per gallon, horsepower, and more. It’s frequently used for regression analysis.
    1. ChickWeight Dataset: This dataset contains information about the weights of chickens at different time points and under different diets. It’s commonly used for longitudinal data analysis.
    1. AirPassengers Dataset: This dataset represents the number of international airline passengers over several years. It’s widely used for time series analysis.
    1. CO2 Dataset: This dataset contains measurements of CO2 uptake by different plants. It’s used for experiments involving plants and environmental factors.
    1. USArrests Dataset: This dataset provides crime statistics for U.S. states, including murder rates, assault rates, and more. It’s often used for crime analysis and visualization.
    1. swiss Dataset: This dataset contains socioeconomic data for the regions of Switzerland. It’s used for regional and multivariate analysis.
  1. BOD Dataset: This dataset contains measurements of dissolved oxygen in the water at different time points. It’s often used for simple time series analysis.
    1. faithful Dataset: This dataset contains information about the eruption times and waiting times between eruptions of the Old Faithful geyser in Yellowstone National Park. It’s commonly used for analyzing waiting times.
    1. women Dataset: This dataset provides data on the average heights and weights of women in a hypothetical population. It’s used for simple statistical demonstrations.

These datasets are valuable for learning and practicing data analysis techniques in R, and they cover a wide range of data types and structures, making them suitable for different types of analyses and exercises.

These datasets are readily available for use by loading them into your R session using the data() function.

data(iris)

Packages in R

So far in our labs, we have been using various functions such as ‘mean()’, ‘hist()’, and ‘median()’, and datasets such as ‘iris’. All of these functions and datasets are readily available when you install R and RStudio. There are, however, more functions and datasets that can be installed. These come in what are called packages.

Installing and loading a new package in R is a two-step process:

  • Step 1: You must install the package. This is done with the install.packages() function. This only needs to be done once.

  • Step 2: You must load in the package. This is done with the library() function. This needs to be done for every new session. That means that if you close and re-open R, you will have to load the package with “library()” again. It also means that you need to put it somewhere in your code chunk before you use it when making an RMarkdown file.

R has dozens of packages, each of which are generally purposed for a particular type of analysis.

Tables and Bar Graphs

R can produce bar graphs, which are closely linked to histograms (though they are reserved for categorical, not quantitative, data).

To produce a bar graph, we first need a frequency table. To produce a frequency table of a categorical variable in R, we use the table() function.

Let’s consider the dataset “mtcars” available in the R package “datasets”. The data was extracted from the 1974 Motor Trend US magazine, and comprises fuel consumption and 10 aspects of automobile design and performance for 32 automobiles (1973–74 models).

data(mtcars)

# More details about the mtcars dataset can be found in R Help. Run the following command in Console
# ?diamonds

Column 2 in “mtcars” is named “cyl” and it indicates the number of cylinders in each car. Let’s say we want to get a barplot for “cyl” to see how many cars are there with different numbers of cylinders.

  • Step 1: We will use table() to create a frequency table of the “cyl”
# Option 1
Num_Cyl <- mtcars$cyl
table(Num_Cyl)
## Num_Cyl
##  4  6  8 
## 11  7 14
# Option 2
table(mtcars$cyl)
## 
##  4  6  8 
## 11  7 14
  • Step 2: Make this into a bar graph using the function “barplot()”
barplot(table(mtcars$cyl))

We can use the “main”, “xlab”, “ylab”, and “col” arguments as in “hist()” function to make a prettier graph.

barplot(table(mtcars$cyl), main = "Number of cylinders - mtcars", 
        xlab = "Number of cylinders", ylab = "Counts", col = "lightskyblue")

Boxplots

R can be used to create boxplots, and the function to create the plots is boxplot(). To make a simple quantile boxplot in R, we can enter boxplot(x, range = 0), where “x” is the dataset that we are interested in.

Before we continue, let’s import our next dataset. We will be working with the dataset called “Quizzes”. It consists of the grades for a class of 171 students across four quizzes, each scored out of 10.

Quizzes <- read.csv("Quizzes.csv")
head(Quizzes)
##   Quiz.1 Quiz.2 Quiz.3 Quiz.4
## 1    8.0    3.5    1.5    7.0
## 2    4.0    5.0    2.0    5.5
## 3    9.0    8.0    8.0    9.5
## 4    9.0    6.5    9.0    6.5
## 5    5.0    6.0    5.5    8.0
## 6    5.5    8.0    4.0    4.0

Let’s create a boxplot of the Quiz 1 grades:

boxplot(Quizzes$Quiz.1, range = 0)

Like in hist(), we can use the “main”, “xlab”, “ylab”, and “col” arguments to make a prettier graph. We can also add the argument horizontal = TRUE to make this into a horizontal boxplot. If we leave this argument out, it will stay as a vertical boxplot.

boxplot(Quizzes$Quiz.1, range = 0, horizontal = TRUE, main = "Quiz 1 Grades", 
        xlab = "Grades", col = "lightsteelblue1")

If we instead want to make a outlier boxplot, we can remove the argument “range = 0”.

boxplot(Quizzes$Quiz.1, horizontal = TRUE, main = "Quiz 1 Grades", 
        xlab = "Grades", col = "lightsteelblue1")

The Quizzes dataset consists of the grades of students across four quizzes. In order to compare the grades of the four quizzes, we can make a side-by-side boxplot by specifying all of the boxplots in order.

boxplot(Quizzes$Quiz.1, Quizzes$Quiz.2, Quizzes$Quiz.3, Quizzes$Quiz.4)

To give each boxplot a name, you can use the names argument. Note that you will need to supply a vector of names here.

boxplot(Quizzes$Quiz.1, Quizzes$Quiz.2, Quizzes$Quiz.3, Quizzes$Quiz.4, 
        names = c("Quiz 1", "Quiz 2", "Quiz 3", "Quiz 4"))

We can specify some more arguments to make the side by side boxplot more informative.

boxplot(Quizzes$Quiz.1, Quizzes$Quiz.2, Quizzes$Quiz.3, Quizzes$Quiz.4, 
        names = c("Quiz 1", "Quiz 2", "Quiz 3", "Quiz 4"), 
        main = "Quiz Grades", ylab = "Grades")

Note:

In R, you can create side-by-side boxplots using the tilde sign (~) to specify the formula for plotting multiple variables. Here’s an example:

# Sample data
set.seed(123)
data <- data.frame(
  Group = rep(c("A", "B"), each = 50),
  Value = c(rnorm(50, mean = 10, sd = 2), rnorm(50, mean = 12, sd = 2))
)

# Create side-by-side boxplot
boxplot(Value ~ Group, data = data,
        main = "Side-by-Side Boxplot",
        xlab = "Group",
        ylab = "Value")

Time plots

We can use R to create timeplots. To create a timeplot (specifically, a line graph), we use the function plot().

The plot() function in R is a versatile and fundamental tool for creating various types of plots and visualizations. It is used to generate a wide range of plots, including scatter plots, line plots, bar plots, histograms, box plots, and more.

The function plot takes two arguments: the x-axis, and the y-axis.

In order to demonstrate we will import the GDP dataset, which contains Canada’s yearly GDP measurements from 1960 to 2021, in billions of dollars.

GDP <- read.csv("GDP.csv")

Let’s create a timeplot of this dataset.

plot(GDP$Year, GDP$GDP)

By default, the “plot()” function simply put dots for each data point. To instead use lines, we will specify the argument type = “l”.

plot(GDP$Year, GDP$GDP, type = "l")

We can specify the “lwd” parameter to set the width of the line, and the “lty” parameter to change the type of the line.

plot(GDP$Year, GDP$GDP, type = "l", lwd = 3, lty = 2)

We can use the “main”, “xlab”, “ylab”, and “col” arguments to add more details to this graph.

plot(GDP$Year, GDP$GDP, type = "l", lwd = 3, lty = 2, 
     main = "GDP of Canada", xlab = "Year", ylab = "GDP ($Billions)", 
     col = "mediumvioletred")
